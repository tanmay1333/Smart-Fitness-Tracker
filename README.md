# Smart Fitness Tracker

## 📌 Description

An Object-Oriented Java application that allows users to track workouts, calculate calories, and store data between sessions.

## 🚀 Features

* Add Cardio and Strength workouts
* Calculate total calories burned
* Menu-driven console system
* Data persistence using file handling

## 🧠 OOP Concepts Used

* Abstraction (Workout class)
* Inheritance (CardioWorkout, StrengthWorkout)
* Polymorphism (calculateCalories method)

## 💾 Data Storage

* Workouts are saved in `workouts.txt`

## 🛠 Technologies

* Java
* OOP
* File Handling
